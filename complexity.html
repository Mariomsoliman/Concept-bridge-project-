What is time complexity?

Time complexity describes how the number of steps an algorithm needs grows with the size of its input.

What is space complexity?

Space complexity tracks how much extra memory an algorithm needs as the input grows.

Why efficiency matters

Efficient algorithms save time and memory. This becomes important when working with large datasets or limited devices like phones or microcontrollers.

Common examples: O(1), O(n), O(n^2)

Connection to problem solving

When you design an algorithm, you are choosing a strategy for solving a problem. Thinking about time and space complexity helps you compare different strategies and pick one that will still work when the input grows.
